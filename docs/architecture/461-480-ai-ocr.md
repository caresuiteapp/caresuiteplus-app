# CareSuite+ — AI & OCR (WP 461–480)

## Umfang

Plattform-UI für OCR- und KI-Jobs auf Basis der Provider-Registry (WP 017).

## Funktionen

| Bereich | Feature |
|---------|---------|
| OCR | Job-Liste, Detail mit extrahiertem Text, Retry bei Fehler |
| KI | Job-Liste mit Zusammenfassungen (Demo) |
| Hub | `/business/platform` — Einstieg OCR + KI |

## Sicherheit

- Keine API-Keys im Frontend
- Jobs referenzieren `secretReference` (Vault)
- Live-Verarbeitung über Edge Functions (vorbereitet)

## Berechtigungen

- `platform.ocr.view` / `platform.ocr.manage`
- `platform.ai.view` / `platform.ai.manage`
