# CareSuite+ — Office Mitarbeitende (WP 181–200)

## Umfang

Vertiefung Mitarbeitenden-Verwaltung: Detail + Bearbeitung.

## Routen

- `/office/employees` — Liste (bestehend)
- `/office/employees/[id]` — Detail mit „Bearbeiten“-Button
- `/office/employees/[id]/edit` — Stammdaten bearbeiten

## Berechtigungen

- `office.employees.view` (bestehend)
- `office.employees.edit` (neu, Admin/Manager)
