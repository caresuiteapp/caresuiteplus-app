# CareSuite+ — Kataloge (WP 441–460)

## Umfang

Mandantenfähige Stammdatenkataloge für Leistungen, Tarife und Qualifikationen.

## Module

| Katalog | Typ | Demo-Positionen |
|---------|-----|-----------------|
| Leistungskatalog Assist | `service` | AB-001 … AB-003 |
| Tarife Abrechnung | `tariff` | PK-SGBXI-45, PK-SGBV-37 |
| Qualifikationen Team | `qualification` | Q-PFK, Q-ERSTHILFE |

## Routen

- `/office/catalogs` — Liste
- `/office/catalogs/[id]` — Detail mit Positionen

## Berechtigungen

- `office.catalogs.view`
- `office.catalogs.edit`

Verknüpfung: Abrechnung → Button „Leistungskatalog“.
