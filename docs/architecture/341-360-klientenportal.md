# CareSuite+ — Klient:innenportal (WP 341–360)

## Umfang

Vertiefung des Klient:innen- und Angehörigenportals analog zum Mitarbeiterportal.

## Funktionen

| Bereich | Feature |
|---------|---------|
| Termine | Liste → Detail mit Betreuungsperson, Hinweisen, Änderungsanfrage |
| Nachrichten | Liste → Detail, lesen, antworten |
| Dokumente | Liste → Detail, Download (Demo) |
| Profil | Stammdaten, KPIs, nächster Termin, Pflegeplan-Zusammenfassung |

## Berechtigungen (`portal.client.*`)

Rollen `client_portal` und `family_portal` erhalten alle Client-Portal-Rechte.

## Demo-Zugang

- Klient:in: `/auth/client-login` → Helga Schneider (`client-001`)
- Angehörige:r: gleiche Datenbasis über `family_portal`
